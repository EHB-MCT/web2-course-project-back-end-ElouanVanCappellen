# Chain Line ‚Äî Back-end API

This repository contains the back-end of **Chain Line**, a full-stack web application inspired by underground alleycat cycling races.  
The back-end provides a REST API for user authentication, route creation, route retrieval, and community interactions such as liking routes.

The API is built with **Node.js**, **Express**, and **MongoDB**, and is designed to be consumed by a separate front-end application.

Live API: https://web2-course-project-back-end-f2er.onrender.com

---

## Up & running

Create a `.env` file in the root of the project:

```env
MONGODB_URI=your_mongodb_connection_string_here
DB_NAME=THE_NAME_OF_YOUR_DB_GOES_HERE

```

## API overview üîÅ

### Auth
- `POST /login` ‚Äì Log in a user
- `POST /register` ‚Äì Register a new user

### Routes
- `GET /routes` ‚Äì Get all routes (supports sorting & filtering)
- `GET /routes/:id` ‚Äì Get a single route by ID
- `POST /routes` ‚Äì Create a new route
- `PUT /routes/:id` ‚Äì Update a route
- `DELETE /routes/:id` ‚Äì Delete a route
- `POST /routes/:id/like` ‚Äì Like a route
- `DELETE /routes/:id/like` ‚Äì Unlike a route
